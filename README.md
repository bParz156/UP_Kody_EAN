## Opis
Powyższy kod stanowi źródło programu do sczytywania kodów EAN13. Program napisany został w ramach laboratorium z kursu Urządzenia Peryferyjne.
Do prawidłowego działania programu potrzebna była funkcja sprawdzająca poprawność wpisywanego kodu EAN.
Funkcja checkSum() sprawdza poprawność cyfry kontrolnej za pomocą algorytmu opisanego we wstępie. Kod na zajęciach działał prawidłowo.
## Technologie
Program został napisany w języku C# z wykorzystaniem biblioteki BarcodeLib.
